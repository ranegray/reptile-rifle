# GOLEM

**Humanoids that Learn Real Work.**

Marketing website for GOLEM, a robotics company deploying intelligent humanoids into manufacturing lines.

## Getting Started

1.  **Install dependencies:**

    ```bash
    npm install
    ```

2.  **Run the development server:**

    ```bash
    npm run dev
    ```

3.  **Open the browser:**
    [http://localhost:3000](http://localhost:3000)
